# Калькулятор Облачной АТС — билайн бизнес

Пошаговый веб‑калькулятор стоимости услуги **«Облачная АТС»** билайн. Самодостаточная страница (`index.html`) без сборки и зависимостей — открывается двойным кликом или хостится как статика.

## Возможности

- **5 шагов:** тариф → опции АТС → аналитика → доп. сервисы → итог.
- Все услуги и цены из прайса **01‑26**: 4 тарифных плана, номера/сотрудники/линии/каналы сверх пакета, запись, CRM, API, коллтрекинг, МультиТрекинг, МультиЧат, SMS‑визитка, речевая аналитика, голосовые кампании, голосовой робот, Колл‑центр PRO.
- **Живой расчёт** справа и липкая панель итога на мобильных.
- **Иконки‑подсказки** у каждой услуги — простым языком, зачем это бизнесу.
- **Сохранение прогресса** (localStorage): закрыл и открыл страницу — продолжаешь с места.
- **Печать / PDF** — итог оформляется как фирменное коммерческое предложение.
- Адаптивная вёрстка, фирменный стиль билайн, шрифт Inter, иконки Lucide.
- OG‑превью для красивого шеринга в мессенджерах.

## Запуск

Открыть `index.html` в браузере. Для корректного OG‑превью разместить на хостинге (например, GitHub Pages) и при необходимости поправить абсолютные URL в `og:*` мета‑тегах.

## Структура

```
index.html               — калькулятор (HTML + CSS + JS в одном файле)
assets/                   — логотип, фавикон, иконки, og-image
прайс оатс.pptx           — исходный прайс (источник цен)
```

## Дисклеймер

Расчёт носит информационный характер и не является публичной офертой. Цены — с НДС (кроме Голосового робота и Колл‑центра PRO), без надбавки 1,2%. Источник: прайс «Облачная АТС» билайн (01‑26).
